**برومبت شامل لمشروع EchoShop**

### الهدف من المشروع:
إنشاء متجر إلكتروني متكامل (EchoShop) مشابه لمنصة أمازون، يدعم لغتين: العربية والإنجليزية، مع مراعاة التصميم الجذاب، تجربة المستخدم الممتازة، وإمكانية التحكم والتخصيص الكامل من لوحة التحكم (Admin).

### تفاصيل المشروع:

#### الصفحات الرئيسية:
- **الصفحة الرئيسية (index.html / index-ar.html):**
  - شعار المتجر "EchoShop" باللون (#a3e635 أخضر فاتح نيون).
  - نافذة بحث عن المنتجات.
  - أزرار: تسجيل دخول (Login)، إنشاء حساب (Sign Up)، مساعدة (Help)، تغيير اللغة (Arabic/English).
  - أيقونة سلة المشتريات متحركة (لونها أخضر فاتح نيون) مع عداد المنتجات.
  - رسالة ترحيب ديناميكية باسم المستخدم عند تسجيل الدخول.
  - زر Logout يظهر فقط عند تسجيل الدخول بلون أحمر.

- **صفحة تسجيل الدخول (login.html):**
  - نموذج لتسجيل الدخول (البريد الإلكتروني، كلمة المرور).
  - ربط فعلي مع Firebase.

- **صفحة إنشاء الحساب (signup.html):**
  - نموذج إنشاء حساب (اسم المستخدم، البريد الإلكتروني، كلمة المرور، تأكيد كلمة المرور).
  - اختيار نوع المستخدم (Client, Seller).
  - ربط فعلي مع Firebase.

- **صفحة السلة (cart.html):**
  - تعرض المنتجات التي تمت إضافتها من المستخدم.

- **صفحة المساعدة (contact.html):**
  - نموذج للتواصل مع خدمة العملاء أو الدعم الفني.

#### لوحة التحكم (صفحة الأدمن admin-dashboard.html):
- تسجيل دخول منفصل للأدمن.
- تحكم كامل بالموقع:
  - تغيير وتعديل اللوجو والصور والبنرات.
  - تغيير أرقام التواصل (مثل رقم الواتس اب للدعم).
  - التحكم في الحسابات المسجلة (تفعيل، تعطيل، حذف).
  - مراقبة الطلبات والمبيعات.

### مزايا وتجربة المستخدم:
- **تفاعلات (UX/UI):**
  - تحسين تجربة المستخدم.
  - صور عالية الجودة مع Lazy Loading.
  - تأثيرات الحركة (Scroll Animation).
  - Tooltip توضيحي للسلة والأزرار.
  - إشعارات تفاعلية عند إضافة المنتجات إلى السلة.

- **توصيات وصفحة مخصصة:**
  - نظام توصيات يعرض منتجات بناءً على اهتمامات المستخدم وتفضيلاته.

- **محرك بحث ذكي:**
  - إمكانية البحث مباشرة عن المنتجات في المتجر.

### تفاصيل تقنية وتنظيم الملفات:
- تنظيم الملفات بشكل هرمي واضح:
```
EchoShop
├── index.html
├── index-ar.html
├── login.html
├── signup.html
├── cart.html
├── contact.html
├── admin-dashboard.html
├── assets
│   ├── images
│   ├── css
│   └── js
└── firebase
```

### الخصائص التقنية:
- **الواجهة الأمامية:** HTML, CSS, JavaScript, Tailwind CSS.
- **الواجهة الخلفية:** Firebase Auth وFirestore.
- **الاستضافة:** Firebase Hosting.

### خواص إضافية مقترحة:
- نظام تقييم ومراجعات للمنتجات.
- ميزة الدردشة المباشرة بين المستخدم وخدمة العملاء.
- نظام تتبع للطلبات.
- دعم الدفع عند الاستلام، مع إمكانية إضافة وسائل دفع إلكترونية لاحقًا.

### ضمانات التنفيذ:
- تسليم كامل للمشروع مع شرح مُفصّل خطوة بخطوة.
- توفير معاينة مستمرة خلال التطوير لتجنب الأخطاء وضمان مطابقة التصميم والوظائف.
- إمكانية التعديل والتخصيص الكامل لكل المكونات من لوحة التحكم.

هذا البرومبت شامل ومتوافق تمامًا مع متطلباتك السابقة وأفكارك، مع إضافة اقتراحات تحسينية تساعدك في تطوير مشروع احترافي، قوي، وجاهز للاستخدام.
